# Brutalist Editorial Portfolio

A bold, editorial-inspired portfolio template built with **Astro** and **Tailwind CSS v4**.

![Brutalist Editorial](https://img.shields.io/badge/Astro-6.x-FF5D01?logo=astro)
![Tailwind](https://img.shields.io/badge/Tailwind-4.x-06B6D4?logo=tailwindcss)
![License](https://img.shields.io/badge/License-MIT-green)

## ✦ Design Philosophy

**Brutalist Editorial** draws from magazine layout grids, Swiss typography, and raw industrial aesthetics. Massive display type, asymmetric compositions, acid-green accents, and a noise-textured dark/light palette — designed to be **unforgettable**.

## 🚀 Quick Start

```bash
pnpm install
pnpm dev
```

Open [http://localhost:4321](http://localhost:4321) to see your site.

## 🎨 Customization

1. Edit `src/pages/index.astro` — change `name`, `role`, `tagline`
2. Edit `src/components/Projects.astro` — swap in your projects
3. Edit `src/components/Skills.astro` — add your skills
4. Edit `src/components/Contact.astro` — update email and links
5. Tweak colors in `src/styles/global.css` — search for `--color-`

## 🏗️ Build

```bash
pnpm build
pnpm preview
```
